# 五子棋專題競賽
## 五子棋介紹：
本次五子棋專題競賽的規則為21*21的大小
  
**Standard Gomoku规则：**
- 黑子先行，黑白双方轮流落子。
- 首先在横、竖、斜方向上成五（连续五个己方棋子）者为胜。
- 超过五子以上不算赢也不算输。
- 若棋盘下满，或对局双方达成协议即为和局。
- **禁止先下的黑棋**下出*双活三、双四、长连（超过五子以上的连线）*，下出则判败（有附上禁手規則）
